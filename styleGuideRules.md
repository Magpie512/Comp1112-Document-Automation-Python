# STYLE GUIDE

### camalCase
Use **camelCase** (start with lowercase, capitalize subsequent words)
	- Example: `myVariableName`

### Meaningful Variable Names
All variable names must be meaningful
    - Example: 'age' or 'destinationPointer'

### Header
Start each program with a header containing:
	- Name
	- Student Number
	- Date
	- Time
	- Brief description

### Sectional Comments
Above every code block should be a comment outlining it's function.
